\chapter{Conceituação das soft skills}

Neste capítulo, explicamos a importância das soft skills que escolhemos para esta pesquisa, as quais foram listadas no Capítulo \ref{cap:research}. Aqui discutimos suas aplicações na profissão do programador de software e propomos um mapeamento com os traços de personalidade descritos pelo Modelo dos Cinco Fatores. O objetivo deste capítulo é esclarecer o significado de cada soft skill e encontrar informações suficientes para desenvolver formas de identificá-las automaticamente.
